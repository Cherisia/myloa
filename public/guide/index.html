myloa 가이드
Step 1

시작하기

myloa는 Discord 계정으로 로그인합니다. 별도 회원가입이 없어요.

1 myloa.app 접속

접속하면 데모 대시보드가 바로 보입니다. 로그인 없이도 화면을 미리 볼 수 있어요.

2 Discord로 로그인

우측 상단 로그인 버튼 클릭 → Discord 인증 창에서 승인을 누르면 완료됩니다.

Step 2

캐릭터 추가

로스트아크 API 키를 연결하면 내 캐릭터를 자동으로 불러옵니다.

1 API 키 발급
1.developer.lostark.game.onstove.com 접속 후 게임 계정으로 로그인
2.[나의 애플리케이션] → [새 애플리케이션 등록]
3.이름을 입력하고 등록하면 API 키가 발급됩니다
myloa 안의 [API 키 발급 가이드] 버튼을 누르면 단계별 안내를 바로 볼 수 있어요.
▭
스크린샷 — 로스트아크 개발자 포털 API 키 화면
2 캐릭터 검색 및 추가

대시보드에서 [+ 캐릭터 추가]를 클릭하면 모달이 열립니다.

1.API 키 붙여넣기
2.캐릭터 이름 검색 → 원정대 목록 확인
3.추가할 캐릭터 선택 후 [추가] 클릭
3 순서 변경 및 삭제

[캐릭터 편집]에서 드래그 앤 드롭으로 순서를 바꾸거나 삭제할 수 있습니다.

Step 3

레이드 설정

어떤 레이드를 할지, 골드 수령 여부를 캐릭터별로 설정합니다.

1 자동 세팅 (권장)

대시보드 우측 상단 [자동 세팅]을 클릭하면 아이템레벨 기준으로 적합한 레이드를 자동 추천합니다.

1.전략 선택 (골드 최대화 / 빠른 클리어 등)
2.미리보기 확인 후 [적용]
▭
스크린샷 — 원정대 자동 세팅 모달
2 수동 설정

각 캐릭터 카드의 [레이드 설정]에서 세부 조정이 가능합니다.

·레이드 및 난이도 (노말 / 하드) 선택
·골드 체크 여부 설정 — 캐릭터당 최대 3개
·EX 레이드는 원정대 전체에서 1캐릭터만 선택 가능
Step 4

숙제 체크

클리어한 레이드 관문을 클릭해서 체크합니다.

1 관문 클리어 체크

대시보드에서 각 레이드 칸을 클릭하면 체크가 됩니다. 레이드 전체 클리어 시 카드가 완료 상태로 바뀝니다.

2 주간 자동 초기화

레이드 체크는 매주 수요일 오전 6시에 자동으로 초기화됩니다. 별도 조작이 필요 없어요.

Step 5

골드 확인

이번 주 골드 수익을 대시보드에서 바로 확인할 수 있습니다.

1 상단 골드 요약

대시보드 상단에 이번 주 총 예상 골드 수익이 표시됩니다. 레이드를 클리어할수록 실시간으로 올라갑니다.

2 캐릭터별 골드 배지

각 캐릭터 카드 하단에 해당 캐릭터의 주간 골드 수령량이 표시됩니다.

3 더보기 골드 설정

일부 레이드에서 귀속 아이템 선택 시 추가 골드를 받을 수 있습니다. 레이드 칸을 우클릭하면 옵션이 나옵니다.

1.더보기 골드 받기 활성화
2.귀속 / 거래 여부 선택
▭
스크린샷 — 더보기 골드 옵션 메뉴
Step 6

커스텀 숙제

레이드 외에 나만의 일일·주간 숙제를 직접 추가할 수 있습니다.

1 항목 추가

[커스텀 숙제 편집]에서 새 항목을 만듭니다.

1.숙제 이름 입력
2.아이콘 선택
3.초기화 주기 선택 — 일일 / 주간
4.[추가] 클릭 → 대시보드에 바로 반영
2 체크 및 초기화

추가한 항목은 각 캐릭터 카드에서 체크할 수 있고, 설정한 주기에 따라 자동 초기화됩니다.

Step 7

원정대

길드원과 레이드 현황을 공유하기 위한 공간입니다.

1 원정대 만들기

상단 메뉴 [원정대] → [원정대 만들기] 클릭

1.원정대 이름 입력 후 생성
2.자동 발급된 초대 코드를 길드원에게 공유
2 초대 코드로 참가
1.[원정대] 페이지 → [초대 코드로 참가]
2.코드 입력 후 신청
3.리더 수락 → 멤버 합류 완료
3 멤버 관리 (리더)

[설정] 탭에서 가입 신청 수락·거절, 역할 변경, 강퇴를 관리할 수 있습니다.

Step 8

현황 공유

원정대 멤버들의 레이드 클리어 현황을 실시간으로 확인합니다.

1 레이드 공개 설정

내 현황이 다른 멤버에게 보이려면 공개 설정이 필요합니다.

1.우측 상단 프로필 → [설정]
2.[레이드 현황 공개] 토글 ON
끄면 원정대 멤버에게도 현황이 숨겨집니다.
▭
스크린샷 — 설정 페이지 레이드 공개 토글
2 원정대 레이드 현황

원정대 상세 페이지 [레이드 현황] 탭에서 멤버별 클리어 진행 상황을 확인합니다.

·레이드별 클리어 멤버 수 표시
·레이드 행 클릭 → 멤버별 상세 펼치기
3 그룹 (지인 공유)

소수의 지인과 공유하려면 [그룹] 탭에서 닉네임으로 검색 후 그룹 요청을 보내면 됩니다. 수락하면 서로의 현황을 볼 수 있어요.
